A fresh start

04.06.2017

Starting this blog marks the beginning of a new chapter of my life. A lot of things are about to change all at the same time, so it feels like a good time to start writing some things down.

Back to the future

I love learning new skills and am passionate about developing my coding skills in particular. So that will be a particular focus. I am planning to use this blog as a platform to document my progress and share some of what I end up learning.

As I start to develop a small portfolio of projects, I will also write about that here.

An experiment in static site generation

Aside from the content of the blog, I am also quite excited about using the blog as a way of learning more about static site generation. I used Hexo as a tool to create the blog deploying it via Github pages. As I go along I am planning to tweak the platform and add more functionality.
